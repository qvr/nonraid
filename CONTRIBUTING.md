## How to contribute to NonRAID project

### If you have a question
If you have a question about the NonRAID project, please check our [README](README.md) first. If your question is not answered there, you can open a new discussion on our [GitHub Discussions page](https://github.com/qvr/nonraid/discussions).

### If you found a bug
If you found a bug in the NonRAID project, please open a new issue on our [GitHub Issues page](https://github.com/qvr/nonraid/issues).

### Use of AI tools
When contributing to this repository, you are allowed to use AI tools such as Claude Code, Github Copilot etc to assist you in writing code. However, please ensure that you personally understand all code and content generated by AI tools before submitting it.

Do not use AI tools to generate the descriptions of pull requests or issues you report. All issue descriptions should be written in your own words demonstrating your understanding of the problem and solution, and to keep the description clear and concise.

When submitting contributions that include content generated by AI tools, please include a note in your pull request or issue description indicating which parts were generated by AI. This helps maintain transparency and allows reviewers to give appropriate attention to those sections.
